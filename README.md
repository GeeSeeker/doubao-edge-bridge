# 🛡️ Doubao Edge Bridge (豆包-Edge 桥梁)

> 一款基于 Manifest V3 的浏览器扩展，自动将非白名单域名重定向至 Microsoft Edge 浏览器打开，帮助你轻松管理可信站点。

---

## ✨ 功能特性

### 🔒 核心功能：域名拦截与重定向

- **智能拦截**：自动检测当前页面域名是否在白名单中。
- **无缝跳转**：**非白名单域名**将通过 `microsoft-edge:` 协议重定向至 Edge 浏览器打开。
- **自动清理**：可配置重定向后自动关闭原标签页，保持浏览器整洁。
- **子域名支持**：支持子域名通配（如白名单包含 `example.com`，则 `sub.example.com` 同样放行）。
- **智能过滤**：自动忽略 `chrome:`、`edge:`、`about:`、`devtools:` 等特殊协议。

### 📝 白名单管理

- **添加域名**：支持直接输入域名，自动归一化处理（去除 `http://` 等前缀）。
- **智能排序**：
  - 基于**核心词 (SLD)** 和 **层级** 的智能排序算法。
  - 同属一个组织的域名（如 `google.com` 和 `google.co.uk`）会自动聚在一起。
  - 主域名（如 `baidu.com`）优先于子域名（如 `tieba.baidu.com`）。
  - IP 地址单独归类。
- **编辑/删除**：行内编辑，一键移除。
- **搜索过滤**：实时搜索，快速定位。
- **批量导入**：支持从 `.txt` / `.csv` / `.json` 文件批量导入域名。

### 🚀 高级功能

- **最近拦截记录**：展示最近 5 条被拦截的域名记录，支持一键复制 URL。
- **备份系统**（最多 3 个备份）：
  - 📦 **创建备份**：支持自定义名称。
  - ⭐ **收藏备份**：重要备份置顶显示。
  - 🔄 **恢复备份**：一键覆盖当前白名单（带二次确认）。
  - 📊 **差异对比**：可视化展示当前白名单与备份的差异（新增/缺失）。
  - 📥 **导出备份**：下载为 `.txt` 文件。
  - ✏️ **在线编辑**：直接在弹窗中修改备份内容。
  - ⚠️ **自动轮替**：超出上限时提示删除最早的非收藏备份。

### ⚙️ 个性化设置

- **自动关闭标签**：开关控制重定向后是否自动关闭原标签。
- **主题模式**：支持 **浅色** / **深色** / **跟随系统** 三种模式。
- **数据重置**：一键恢复默认设置。

## 📦 安装方法

### 方式一：下载源码加载 (推荐)

1. **获取代码**

   ```bash
   git clone https://github.com/GeeSeeker/doubao-edge-bridge.git
   ```

   或者直接点击 GitHub 页面右上角的 `Code` -> `Download ZIP` 并解压。

2. **打开浏览器扩展管理页面**

   在豆包浏览器地址栏输入 `chrome://extensions/`

3. **启用开发者模式**

   打开页面右上角的 **"开发人员模式"** (Developer mode) 开关。

4. **加载扩展**

   点击左上角的 **"加载已解压的扩展程序"** (Load unpacked) 按钮，选择本项目所在的文件夹（即包含 `manifest.json` 的目录）。

5. **完成** 🎉

   扩展图标将出现在浏览器工具栏中，点击即可使用。

## 🚀 使用说明

### 默认白名单

安装后，插件预置了以下常用域名：

- `doubao.com`
- `chatgpt.com`
- `claude.ai`
- `m365.cloud.microsoft`
- `gemini.google.com`
- `live.com`

### 常见操作

1. **启用/禁用**：点击弹出面板顶部的开关即可一键启停。
2. **添加白名单**：在首页输入框输入域名（如 `github.com`），回车或点击添加。
3. **备份数据**：切换到“高级”标签页，输入备份名称点击创建。
4. **查看拦截**：在“高级”标签页顶部可以看到最近被拦截跳转的网址。
